# Code of Conduct

## Our Standards

mosaicraft is an open and welcoming project. We expect all contributors and
participants to act with respect, courtesy, and good faith when interacting
in any project space (issues, pull requests, discussions, code reviews).

In short:

* Be respectful and constructive.
* Assume good intent and ask for clarification before assuming the worst.
* Focus on the technical merits of contributions.
* Accept feedback gracefully and offer feedback kindly.

Behavior that disrupts the project or makes others feel unwelcome is not
acceptable.

## Adopted Standard

This project follows the
[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
which is the de facto standard for open source community conduct. The full
text and translations are available at
<https://www.contributor-covenant.org/>.

## Scope

This Code of Conduct applies within all project spaces, and also when an
individual is officially representing the project in public spaces.

## Enforcement

Instances of unacceptable behavior may be reported privately to the project
maintainer at **hinanohart@gmail.com**. All reports will be reviewed and
investigated promptly and fairly.

The maintainer is obligated to respect the privacy and safety of the
reporter of any incident.
